# 医療費・セルフメディケーション管理 Ver.2

## 内容

このアプリは GitHub Pages で公開できる PWA 対応版です。

### 含まれるファイル

-   index.html
-   manifest.json
-   service-worker.js
-   favicon.ico
-   apple-touch-icon.png
-   icon-192.png
-   icon-512.png
-   icon-1024.png

## GitHub Pages 公開手順

1.  GitHub のリポジトリを開きます。
2.  このフォルダ内のファイルをすべてアップロードします。
3.  Settings → Pages を開きます。
4.  Branch を **main**、Folder を **/(root)** に設定して Save
    を押します。
5.  数分後に公開されます。

## 更新方法

index.html を更新したら GitHub に上書きアップロードしてください。
service-worker.js の CACHE_NAME
を変更するとキャッシュが更新されやすくなります。

## ホーム画面に追加

### iPhone

Safariで開き、共有 → 「ホーム画面に追加」

### Android

Chromeで開き、「インストール」または「ホーム画面に追加」
